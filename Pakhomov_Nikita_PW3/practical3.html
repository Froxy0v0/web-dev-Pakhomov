Opening Developer Console

To open the developer console in Mozilla Firefox, you can press Ctrl + Shift + K (or Cmd + Option + K on macOS). In Safari, you can open the developer console by pressing Option + Command + C.

Choosing Link Types

Generally, the preferable way of including internal links into an HTML page is by using relative links. Relative links are advantageous as they make the website more portable and easier to maintain. However, absolute links are useful in certain situations, such as when linking to resources on external domains or when the structure of the website might change.

Use cases for choosing absolute links include linking to resources hosted on different servers or domains, ensuring the link always points to the correct location regardless of the page's location. Relative links are ideal for links within the same website or domain, simplifying management and allowing for more flexible site restructuring.

Image Formats in Web Development

Three common image formats used in web development are JPG, GIF, and PNG.

  • JPG (or JPEG): This format is best suited for photographs or images with complex color variations. It uses lossy compression, meaning some quality is sacrificed to reduce file size.
  • GIF (Graphics Interchange Format): GIFs are commonly used for animated images and graphics with limited colors. They support transparency and animation but are not ideal for complex images due to limited color depth.
  • PNG (Portable Network Graphics): PNG is a versatile format suitable for various types of images, including graphics and photographs. It supports transparency and lossless compression, making it ideal for web use.

WEBP is an image format developed by Google that offers both lossy and lossless compression. It provides superior image quality and smaller file sizes compared to JPG and PNG.

SVG (Scalable Vector Graphics) is a vector image format that uses XML-based markup. SVG images are resolution-independent and can be scaled without losing quality. They are ideal for graphics like icons, logos, and illustrations.
